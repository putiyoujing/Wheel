# 🎯 幸运转盘 - Lucky Wheel

一个精美的在线转盘游戏生成器，让你可以轻松创建个性化的幸运转盘！

![Preview](https://img.shields.io/badge/Version-1.0.0-brightgreen)
![License](https://img.shields.io/badge/License-MIT-blue)
![HTML5](https://img.shields.io/badge/HTML5-E34F26?logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?logo=javascript&logoColor=black)

## ✨ 功能特点

- 🎨 **9种精美主题**：简约白、暗黑、赛博朋克、森林、海洋、夕阳、糖果、午夜、霓虹
- 🎲 **动态配置**：灵活添加/删除选项，自定义占比
- 🎯 **智能校验**：自动确保占比总和为100%
- 🎮 **流畅动画**：转盘旋转动画、缓动效果、撒花庆祝
- 🔊 **音效系统**：转动音效和胜利音效
- 💾 **独立导出**：生成单文件HTML，可离线使用
- 📱 **响应式设计**：完美适配各种设备

## 🚀 快速开始

### 在线体验
访问 [GitHub Pages](https://putiyoujing.github.io/wheel/) 立即体验！

### 本地运行
1. 克隆仓库：
```bash
git clone https://github.com/putiyoujing/wheel.git
```

2. 进入项目目录：
```bash
cd wheel
```

3. 用浏览器打开 `index.html`：
```bash
# Windows
start index.html

# macOS
open index.html

# Linux
xdg-open index.html
```

## 🎮 使用方法

### 1. 创建转盘
- 输入转盘标题
- 选择喜欢的主题风格
- 添加选项并设置占比（确保总和为100%）

### 2. 生成游戏
- 点击"生成转盘"按钮
- 进入游戏界面

### 3. 开始游戏
- 点击"开始转盘"按钮
- 等待动画结束查看结果

### 4. 导出分享
- 点击"导出游戏"按钮
- 保存为独立HTML文件
- 分享给朋友一起玩！

## 🛠️ 技术栈

- **前端**：HTML5 + CSS3 + JavaScript (ES6+)
- **绘图**：Canvas API
- **动画**：CSS3 Animations + RequestAnimationFrame
- **音效**：Web Audio API
- **样式**：CSS Variables (动态主题切换)
